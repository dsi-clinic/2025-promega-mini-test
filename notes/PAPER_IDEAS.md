## Suggested Secondary Results (Eric, for ML + Bio audience)

### 1. Images vs. Metabolites vs. Combined
- Compare: Images only, Metabolites only, Metabolites + Images
- A convincing negative result (e.g., images add little) is acceptable

### 2. Ablation Study of Metabolites
- Can a comparable model be built with only a subset of metabolites?
- Does adding manually curated imaging features (e.g., organoid size/area) improve performance?
  - If not, this strengthens the negative result from point 1

### 3. Trajectory Input vs. Single Time Point Input
- Compare predicting health from day 3 → day X vs. day X alone
- Trajectory does **not** outperform single time point — suggests days 3 to X-1 add little value
- Recommendation: mention in 1–2 sentences; defer to Stevens on significance
